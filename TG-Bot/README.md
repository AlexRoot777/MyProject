# Bot-TG
123123

Телеграм-бот на Python для продажи MTProto-прокси:

- выдаёт пользователю MTProto-ключ и tg:// ссылку подключения;
- имеет базовую админ-панель;
- хранит пользователей и выданные ключи в SQLite;
- умеет генерировать ключи локально или через серверную команду (`PROXY_GEN_CMD`).

## Быстрый старт

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Создайте `.env`:

```env
BOT_TOKEN=ваш_токен_бота
ADMIN_IDS=123456789,987654321
PROXY_HOST=your.server.ip
PROXY_PORT=443
DATABASE_PATH=bot.db
# Опционально: команда для генерации секрета на сервере
# PROXY_GEN_CMD=/usr/local/bin/generate_mtproto_secret
```

Запуск:

```bash
python bot.py
```

## Команды бота

Пользователь:

- `/start`
- `/myid`
- `/get_proxy` — получить новый MTProto-ключ

Админ:

- `/admin` — открыть inline-панель
- `/ban <user_id>` — отключить пользователя
- `/unban <user_id>` — включить пользователя
- `/issue <user_id>` — выдать ключ конкретному пользователю

## Как работает генерация ключей

По умолчанию бот генерирует секрет формата `dd + 32 hex` внутри процесса Python.

Если задан `PROXY_GEN_CMD`, бот выполняет эту команду на том же сервере, где запущен бот, и использует её stdout как секрет. Это позволяет интегрировать ваш текущий скрипт/инфраструктуру выдачи MTProto-ключей.
